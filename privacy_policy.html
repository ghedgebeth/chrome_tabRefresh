Privacy Policy

Tab Auto Refresh
Last updated: May 29, 2025
Developed by Glenn Hedgebeth

1. Data Collection

We do not collect, store, or transmit any personal or sensitive user data.
This extension operates entirely within your local Chrome browser. It does not communicate with any external servers or third parties.

2. Data Usage

The extension only stores the following non-personal data locally:

  • Tab IDs you've enabled for auto-refresh
  • The refresh interval you've set per tab

This data is stored using Chrome’s storage.local API and is only used to manage the extension's functionality. It is not accessible to Glenn Hedgebeth or any third party.

3. Permissions

The extension uses the following permissions solely for its core functionality:

  • tabs: To list open tabs and apply refresh actions
  • storage: To save your tab preferences and intervals
  • scripting: To reload a selected tab using in-tab scripting
  • host_permissions: ["<all_urls>"]: To allow tab refreshing across any website

These permissions do not grant access to your personal information or browsing history.

4. Third Parties

This extension does not use any third-party scripts, trackers, or analytics services.

5. Changes to This Policy

If the privacy policy changes in the future, updates will be posted here and reflected in the extension’s version history. Users are encouraged to review the policy periodically.
